This template loads code into a Strapdown markdown section.

This is a simple template for reading an external file into Strapdown.js xmp/textarea using XMLHttpRequest. It's a general synchrounous way to ready in external files (unlike jQuery load which is asynchrounous AJAX). I added backticks so that it is rendered as syntax highlighted code. Derived from this gist.

I added a head and body which the original didn't have. If you define a title then Strapdown will create the title bar. Hand coded divs etc should be given class='container' so that Bootstrap makes them nice.
